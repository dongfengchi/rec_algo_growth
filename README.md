# 关于我

你好哇~很高兴你想了解我~

我叫董峰池，但我更习惯别人叫我峰池。目前就职于字节跳动，是一名推荐算法工程师，所在团队负责西瓜视频推荐算法工作。

* 一是做出一个自己满意的个性化推荐系统的科普教程。目前正在进行中，可以参考[算法工程师成长路径](https://mp.weixin.qq.com/mp/appmsgalbum?action=getalbum&album_id=1393084303915155457&__biz=MzI3MDE2MzY2NQ==#wechat_redirect)
